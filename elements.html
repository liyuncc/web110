HTML Elements

Tag Description Block, Inline or N/A Container or Empty
<!-- --> A HTML comment tag. It's used to provide additional information about the code and invisible on a browser. N/A Container
<!DOCTYPE> It's a declaration that tells a browser what kind of document a page is. N/A Container
<a> A tag that provides a link from one page document to another. Inline Container
<article> A tag that can stand alone and contain the content independently. Block Container
<aside> A tag that provides additional information about the content surrouding it. It could also use as a sidebar in an article. Block Container
<body> A tag that has all the contents of a document which will present on a browser. Block Container
<br /> A tag that creates a line break. Inline Empty
<dd> A tag that depicts a term from a description list. It's often used with <dl> and <dt>. Block Container
<div> It’s used for grouping content that doesn’t convey semantic meaning, styling purposes (CSS), or for scripting (JavaScript). Block Container
<dl> A tag that presents a description list. Block Container
<dt> A tag that provides definition of a term from a description list. Block Container
<em> A tag that emphasizes a text of a sentence. When a screen reader renders the text wrapped around with this tag, it would change its vocal inflection. Inline Container
<fieldset> A tag that is shown as a box to include related elements together in a form. Block Container
<figcaption> A tag that provides a caption of a figure element. Block Container
<figure> A tag that shows different types of figures. For example, pictures, charts, and diagrams. Block Container
<footer> A tag that provides a footer of a document. For example, contact information and copyright. Block Container
<form> A tag that creates a form for people to type in input. Block Container
<head> A tag that contains several elements, such as a title of a page, styles and meta information. It's not presenting visually on the page. Block Container
<header> It's used when there’s a new section of content with an introductory part. And it provides additional semantic structure of the document. Block Container
<html> It tells a browser that HTML is the language a document is going to use. It includes all the elements of the document, except the <!DOCTYPE>. Block Container
<img /> It inserts images to a web page and has two required attributes (e.g., scr and alt). Block Empty
<input> It allows people to enter input and is usually used with <form> elements. Block Empty
<label> It creates labels for corresponding input elements. Block Container
<li> It shows a list of items. Block Container
<link /> It provides links from a document to external resources. For example, CSS files. It's included in the <head> element. Block Empty
<meta /> It provides information about data of a page. For example, character set, author and decription of the page. It's contained in the <head> element. Block Empty
<nav> It shows a section of a page that links to other pages or other parts within a site. Basically, it’s a section with navigation links. Block Container
<ol> It shows an ordered list. Block Container
<p> It creates a paragraph on a page. Block Container
<script> It provides a script (such as Java Script) within a HTML document. Block Container
<section> It creates a section that contains semantic meaning, and would be listed in the document outline and work together with headings. Block Container
<strong> It presents an important text semantically. Inline Container
<style> It provides styles of a page. Block Container
<table> It means a table on a page. It includes <tr>, <th> and <td> to create a table. Block Container
<td> It means "table data" and allows people to enter data cells. Block Container
<th> It means "table header" and usually shows on the first row as a description about its following columns. Block Container
<title> It presents a title of a page. It's shown as the title of the page in search engines and when added to bookmarks. Block Container
<tr> It means "table row" and creates a row within a table. Block Container
<ul> It creates an unordered list. Block Container
